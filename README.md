Sobre o Projeto

Este projeto foi desenvolvido como Trabalho de Conclusão de Curso do curso de Desenvolvimento de Software da ETEC Irmã Agostina, com o objetivo de criar uma plataforma de ensino online voltada para a área de tecnologia.

A proposta surgiu da necessidade de oferecer um ambiente de aprendizado que priorize a simplicidade e a acessibilidade, permitindo que estudantes de diferentes perfis possam acessar conteúdos de forma intuitiva e organizada.

A plataforma foi planejada para reduzir barreiras de navegação, apresentando páginas com layouts simplificados, menus objetivos e fácil acesso aos recursos disponíveis. Além disso, conta com ferramentas de personalização que permitem ao usuário adaptar a experiência de uso conforme suas preferências.

Objetivos

O principal objetivo do projeto é disponibilizar uma plataforma educacional que facilite o aprendizado na área de tecnologia por meio de recursos acessíveis e organizados.

Entre os objetivos específicos estão:

Oferecer conteúdos em diferentes formatos;
Facilitar o acompanhamento da evolução dos estudantes;
Promover acessibilidade através da personalização visual;
Disponibilizar uma interface simples e intuitiva;
Organizar cursos em trilhas de aprendizado.
Funcionalidades
Cursos em Vídeo e Texto

A plataforma disponibiliza conteúdos em vídeo e materiais textuais, permitindo que cada aluno escolha a forma de estudo mais adequada ao seu perfil.

Trilhas de Aprendizado

Os cursos são organizados em trilhas, possibilitando uma sequência lógica de estudos e um acompanhamento mais eficiente da evolução do aluno.

Sistema de Status

Cada curso possui um status que indica o progresso do estudante:

Não Matriculado
Não Iniciado
Em Andamento
Concluído
Personalização de Interface

Pensando na acessibilidade, a plataforma possui um painel de configurações que permite:

Alteração de cores da interface;
Modificação das fontes utilizadas;
Adaptação visual conforme as necessidades do usuário.
Tecnologias Utilizadas

O desenvolvimento da plataforma foi realizado utilizando os conhecimentos adquiridos durante o curso e as seguintes tecnologias:

MySQL
Desenvolvimento

Durante a construção do projeto foram aplicados conceitos estudados ao longo do curso, incluindo desenvolvimento web, banco de dados, programação, modelagem de sistemas, interface de usuário e boas práticas de desenvolvimento de software.

O projeto representa a integração dos conhecimentos adquiridos pelos integrantes da equipe durante a formação técnica, demonstrando a capacidade de planejar, desenvolver e implementar uma solução tecnológica completa.

Considerações Finais

A plataforma foi desenvolvida com foco em proporcionar uma experiência de aprendizagem acessível, organizada e eficiente para estudantes da área de tecnologia. O projeto também serviu como uma oportunidade para consolidar os conhecimentos obtidos ao longo do curso e vivenciar etapas reais do desenvolvimento de software.
